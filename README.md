<h1 align="center">How to Use 🤔</h1>

````
- Clone this repository:
$ https://github.com/KratosOfficial/Modern-Login-UI

- Enter in directory:
$ cd authenication
- For install dependencies:
$ flutter packages get

- Run the app: 
$ flutter run

````
